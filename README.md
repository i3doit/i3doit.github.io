# 2025 香港跨年电子烟花秀 (Virtual Hong Kong New Year Fireworks 2025)

[![SEO Optimization](https://img.shields.io/badge/SEO-Optimized-brightgreen)](https://i3doit.github.io/MyTools)

本项目是为了纪念 **2025 年 12 月 31 日** 这一特殊时刻而设计的在线交互式烟花系统。在传统烟花表演取消之际，我们通过 HTML5 Canvas 零污染地延续维多利亚港的璀璨传统。

## 🌟 核心功能 (Key Features)
- **多形态烟花系统**：支持点击切换 **心形 (Heart)**、**星形 (Star)**、**传统圆形 (Circle)** 及 **文字烟花 (Text)**。
- **无后端社交链路**：通过 URL 编码技术，支持展示完整的分享者头像链，无需数据库即可实现社交互动。
- **分享卡片生成**：集成 `html2canvas` 自动生成带二维码的跨年祝福卡片。
- **响应式设计**：完美适配移动端和桌面浏览器。
